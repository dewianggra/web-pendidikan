Materi Pembelajaran

Kumpulan materi dasar pemrograman dan ilmu komputer

1. Pemrograman Dasar

Pemrograman dasar adalah fondasi utama dalam dunia pengembangan perangkat lunak.

Konsep Dasar

  • Variabel: Tempat menyimpan data sementara. Contoh: nama = "Andi"
  • Tipe Data: Seperti int, float, string, bool
  • Operator: +, -, ==, &&

Struktur Kontrol

  • If: Pengambilan keputusan: 
    if (nilai > 75) {
                         print("Lulus");
                        }
  • For: Perulangan: 
    for (int i = 0; i < 5; i++) {
                      print(i);}
  • While: Perulangan dengan kondisi: 
    while (x < 10) {
                        x++;}

Fungsi dan Modularisasi

  • Fungsi: Blok kode untuk tugas tertentu.
  • Modularisasi: Memecah program menjadi bagian-bagian kecil.
  • Contoh fungsi: 
    int tambah(int a, int b) {
                         return a + b;}
2. Struktur Data

Struktur Data

  • Array: Struktur data statis yang menyimpan elemen sejenis dalam satu blok memori. Contoh: 
    int angka[5] = {1, 2, 3, 4, 5};
  • Linked List: Kumpulan node yang saling terhubung, masing-masing node menyimpan data dan pointer ke node berikutnya.
  • Stack: Struktur data LIFO (Last In, First Out). Operasi utama: push, pop.
  • Queue: Struktur data FIFO (First In, First Out). Operasi utama: enqueue, dequeue.
  • Tree: Struktur data hierarkis dengan node akar dan cabang. Contoh: binary tree.
  • Graph: Kumpulan simpul (node) dan sisi (edge) untuk merepresentasikan hubungan. Digunakan dalam jaringan, rute, dll.
3. Database

Database

  • Pengertian: Kumpulan data terorganisir yang bisa diakses secara elektronik.
  • DBMS: Sistem yang mengelola database, seperti MySQL, PostgreSQL, Oracle.
  • SQL Dasar: 
    SELECT * FROM siswa;
                            INSERT INTO siswa (nama, umur) VALUES ('Ayu', 17);
                            UPDATE siswa SET umur = 18 WHERE nama = 'Ayu';
                            DELETE FROM siswa WHERE nama = 'Ayu';
  • Relasi Antar Tabel: Menghubungkan tabel satu dengan lainnya menggunakan foreign key (kunci asing).
4. Algoritma

Algoritma

  • Logika dan Alur: Menyusun langkah logis untuk menyelesaikan masalah.
  • Pseudocode: Penulisan logika algoritma secara mirip kode program namun mudah dibaca.
  • Contoh Algoritma:
    • Bubble Sort: Mengurutkan data dengan membandingkan dua elemen berurutan dan menukarnya jika salah urut. 
      for i in range(n):
                                  for j in range(n-i-1):
                                    if data[j] > data[j+1]:
                                    tukar(data[j], data[j+1])
    • Linear Search: Mencari data satu per satu dari awal sampai ditemukan.
